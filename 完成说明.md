## 🎯 接下来的操作步骤：

### 步骤 1：创建 GitHub 仓库

访问 https://github.com/new

```
Repository name: finecourse
Description: FineCourse - Course Management System APK
Public ✅
☑ Initialize this repository with a README
```

点击 **"Create repository"**

---

### 步骤 2：推送代码到 GitHub

在你的终端运行以下命令（替换 `<你的用户名>`）：

```bash
cd /home/harden/.openclaw/workspace/course-newsystem

# 初始化 Git（如果还没初始化）
git init

# 添加所有文件
git add .

# 创建提交
git commit -m "Initial commit: FineCourse APK ready for build"

# 添加远程仓库（替换 <你的用户名>）
git remote add origin https://github.com/<你的用户名>/finecourse.git

# 推送到 GitHub
git push -u origin main
```

---

### 步骤 3：触发构建

1. 访问：`https://github.com/<你的用户名>/finecourse/actions`

2. 点击 **"Build APK"** 工作流

3. 点击 **"Run workflow"** → **"Run workflow"**

4. 等待 5-10 分钟

5. 下载 **"FineCourse-APK"** 工件

---

## 📲 安装到手机

1. 将 APK 下载到手机
2. 允许"安装未知来源应用"
3. 点击 APK 安装

---

## ⚠️ 注意事项

- GitHub Actions 每月有 2000 分钟免费构建时间
- APK 会保存 7 天
- 如果构建失败，可以重试
- 也可以用 Codemagic 作为备用方案
