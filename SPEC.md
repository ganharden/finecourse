# 课程管理与排课系统 - 规格说明书

## 1. 项目概述

- **项目名称**: 精课程管理系统 (FineCourse)
- **项目类型**: Web 全栈应用
- **核心功能**: 课程管理、教师管理、学生管理、教室管理、智能排课
- **目标用户**: 学校教务管理人员

## 2. 技术栈

- **后端**: Node.js + Express
- **前端**: 原生 HTML + CSS + JavaScript (无框架依赖)
- **数据库**: JSON 文件存储
- **端口**: 3001 (API), 8080 (前端)

## 3. UI/UX 规格

### 3.1 整体风格
- **主题**: 深色现代专业风
- **主色调**: #1a1a2e (深蓝黑)
- **强调色**: #e94560 (珊瑚红)
- **辅助色**: #0f3460 (深蓝), #16213e (暗蓝)
- **成功色**: #00d9a5 (青绿)
- **警告色**: #ffc107 (琥珀)
- **文字色**: #eaeaea (主), #a0a0a0 (次)
- **背景**: 渐变叠加几何图案

### 3.2 布局
- 左侧固定导航栏 (240px)
- 右侧主内容区 (flex: 1)
- 顶部标题栏 (60px)
- 卡片式内容展示
- 响应式设计 (最小支持 1024px)

### 3.3 交互
- 按钮 hover 动画 (scale 1.02, 阴影加深)
- 卡片 hover 浮起效果
- 模态框滑入动画
- 列表项渐显加载
- 删除二次确认弹窗
- 成功/失败 Toast 提示

### 3.4 组件
- 导航标签 (带图标)
- 数据卡片 (带统计数字)
- 表格 (斑马纹, hover 高亮)
- 表单 (标签式输入框)
- 按钮 (主/次/危险三色)
- 模态框 (居中, 背景模糊)
- 时间选择器
- 下拉选择器

## 5. 数据量 (中等)

- **课程**: 25 门 (涵盖各学科)
- **教师**: 15 人 (各科目骨干教师)
- **学生**: 120 人 (4个班级)
- **教室**: 20 间 (不同容量)
- **课表**: 80 条 (覆盖周一至周五)

## 6. 功能模块

### 6.1 仪表盘
- 统计卡片: 课程数、教师数、学生数、教室数
- 本周课表概览
- 最近操作记录

### 6.2 课程管理
- 列表展示 (可搜索、筛选)
- 新增课程 (表单验证)
- 编辑课程 (弹窗表单)
- 删除课程 (二次确认)
- 课程详情查看

### 6.3 教师管理
- 卡片/列表双视图
- 新增/编辑/删除
- 授课统计
- 可用时间管理

### 6.4 学生管理
- 分班级展示
- 选课管理
- 新增/编辑/删除

### 6.5 教室管理
- 容量筛选
- 设备标签
- 新增/编辑/删除

### 6.6 排课管理
- 周视图课表
- 手动排课
- 智能排课 (自动检测冲突)
- 冲突提示
- 导出功能

## 7. API 接口

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /api/stats | 获取统计数据 |
| GET/POST | /api/courses | 课程 CRUD |
| GET/POST | /api/teachers | 教师 CRUD |
| GET/POST | /api/students | 学生 CRUD |
| GET/POST | /api/classrooms | 教室 CRUD |
| GET/POST | /api/schedules | 课表 CRUD |
| POST | /api/schedules/auto | 智能排课 |
